# cuscontest-tools

Este repositorio sirve para que puedan crear problemas para los eventos del Cuscontest organizado por el ACM UNSAAC Chapter Cusco.

Inspirado en la librería testlib.h de MikeMirzayanov, pero hecho para facilitar en los procesos de crear para el cuscontest.

Por el momento, solo está diseñado para crear problemas sencillos. Pero está pensando para crear problemas más avanzado como problemas de opciones múltiples o problemas interactivos.
